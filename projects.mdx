**Проект в Releazio** — это основной контейнер, который объединяет все ваши сущности:
* **Платформы** (iOS, Android, Web)
* **Версии** (Release Notes)
* **Посты** (Публикации)

В рамках одного проекта вы можете управлять несколькими платформами одновременно (например, App Store и Google Play для одного приложения).

## Настройки проекта

Чтобы изменить параметры текущего проекта:

1. Откройте нужный проект.
2. В левой панели дашборда найдите блок **УПРАВЛЕНИЕ**.
3. Нажмите на пункт **Настройки проекта**.

### Доступные действия

В этом разделе вы можете выполнить следующие настройки:

<AccordionGroup>
  <Accordion title="Изменить название" icon="pen-to-square">
    Просто введите новое имя в поле ввода названия. Изменения сохраняются автоматически или по кнопке сохранения (в зависимости от интерфейса).
  </Accordion>

  <Accordion title="Узнать ID проекта" icon="fingerprint">
    Поле **ID проекта** содержит уникальный идентификатор.
    * Он используется для работы с API и интеграциями.
    * Рядом с полем есть кнопка копирования для удобства.
    
    <Warning>
      **Важно:** ID проекта **нельзя изменить** после создания. Он генерируется системой один раз. Если вам критически нужен другой ID (например, более красивый), придется создать новый проект.
    </Warning>
  </Accordion>
</AccordionGroup>

## Удаление проекта

Вы можете удалить проект в самом низу страницы настроек.

<Warning>
  **Осторожно: Это необратимое действие!**
  
  Перед удалением убедитесь, что вы сохранили всю важную информацию. При удалении проекта **безвозвратно исчезнут** все вложенные данные:
  * Связанные платформы
  * История версий
  * Все посты и черновики
</Warning>
