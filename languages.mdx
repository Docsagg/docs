Раздел **«Языки»** позволяет адаптировать контент вашего проекта под разную аудиторию. Здесь вы настраиваете список доступных локализаций для [Постов](/posts).

## Добавление языков

Чтобы добавить поддержку нового языка в проект:

<Steps>
  <Step title="Перейдите в настройки">
    В левой панели дашборда найдите блок **УПРАВЛЕНИЕ** и выберите пункт **«Языки»**.
  </Step>
  <Step title="Добавьте язык">
    Нажмите кнопку **«Добавить»**.
  </Step>
  <Step title="Выберите из списка">
    Укажите коды языков, которые вы планируете использовать (например: `RU`, `EN`, `CN`, `DE`).
  </Step>
</Steps>

### Основной язык (Default)

Один из языков в проекте всегда должен быть **Основным**. Он отмечается специальным лейблом в списке.

<Note>
  **Рекомендация:**
  Устанавливайте основным тот язык, на котором говорит большая часть вашей аудитории. Он будет подставляться по умолчанию при создании новых постов и в интерфейсе виджетов.
</Note>

## Использование в постах

Releazio позволяет создавать локализованные версии обновлений для разных групп пользователей.

1. Откройте форму создания [Поста](/posts).
2. В правом верхнем углу найдите выпадающий список **Язык**.
3. Выберите нужную локаль перед заполнением полей.

<Tip>
  **Как перевести пост?**
  
  Чтобы создать новость на другом языке, создайте **новый пост** с тем же смысловым содержанием и картинкой, но выберите **другой язык** в селекторе.
</Tip>

## Советы по локализации

Чтобы ваши пользователи по всему миру оставались довольны:

* **Синхронность:** Старайтесь выпускать посты на всех языках одновременно с релизом версии.
* **Приоритеты:** Ключевые обновления (Major Releases) обязательно должны иметь перевод на основные языки аудитории.
* **Единообразие:** Используйте одинаковую терминологию во всех переводах, чтобы не путать пользователей.
