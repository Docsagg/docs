## 1. Регистрация и вход

<Card title="Перейти в консоль Releazio" icon="rocket" href="https://releazio.com">
  Зарегистрируйтесь или войдите в личный кабинет. Если интерфейс на английском — переключите язык на **Русский** в левом нижнем углу экрана.
</Card>

---

## 2. Создание проекта

<Steps>
  <Step title="Инициация" icon="plus">
    На экране «Мои проекты» нажмите на карточку **«Новый проект»**.
  </Step>
  <Step title="Настройка данных">
    Введите название (например, `MyApp`). Система создаст **Уникальный ID** автоматически — он понадобится разработчикам позже.
  </Step>
  <Step title="Выбор канала">
    Перейдите к разделу **«Канал распространения»**. Выберите платформу, для которой вы создаете проект в первую очередь.
  </Step>
</Steps>

## 4. Интеграция SDK

Это сердце системы. Скопируйте ваш **API-ключ** в разделе «Интеграция» и передайте разработчику.

<Tabs>
  <Tab title="Web / JS">
    Используйте **NPM пакет** для быстрой интеграции виджета обновлений в ваш сайт.
  </Tab>
  <Tab title="Android">
    Подключите библиотеку через **Gradle** (GitHub) для нативных уведомлений.
  </Tab>
  <Tab title="iOS">
    Интеграция через **Swift Package Manager** для работы с App Store.
  </Tab>
  <Tab title="Backend">
    Используйте **REST API** для кастомных решений (Postman документация).
  </Tab>
</Tabs>

<Warning>Без интеграции SDK пользователи не увидят уведомления, даже если версия опубликована.</Warning>

---

## 5. Ваш первый релиз

Весь процесс публикации версии занимает три простых шага:

<Steps>
  <Step title="Информация">
    Укажите версию и выберите тип окна: **Мягкое** (можно закрыть) или **Принудительное** (обязательно для всех).
  </Step>
  <Step title="Контент">
    Напишите **Пост** об изменениях. Добавьте описание функций и, если нужно, скриншоты.
  </Step>
  <Step title="Пуск">
    Проверьте финальную сводку и нажмите **Опубликовать**.
  </Step>
</Steps>

<Note>После публикации вы сможете гибко управлять правилами обновлений в реальном времени.</Note>
