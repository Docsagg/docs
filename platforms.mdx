**Платформа** — это магазин приложений или канал распространения, где опубликован ваш продукт. Releazio позволяет управлять множеством платформ в рамках одного проекта.

## Добавление платформ

Чаще всего платформы добавляются сразу при создании проекта.

<Card title="Как выбрать платформы при старте" icon="rocket" href="/quickstart">
  Инструкция в разделе «Начало работы»
</Card>

Если вам нужно добавить новый магазин позже (например, вы решили выйти в RuStore), следуйте инструкции:

<Steps>
  <Step title="Перейдите к каналам">
    В левой панели дашборда найдите раздел **Каналы распространения**.
  </Step>
  <Step title="Инициируйте добавление">
    Нажмите кнопку **«Добавить платформу»**.
  </Step>
  <Step title="Выберите платформу">
    В открывшемся списке выберите нужный магазин и нажмите **«Продолжить»**.
  </Step>
</Steps>

### Поддерживаемые платформы
Вы можете добавить любые из следующих магазинов:
* **Мировые:** App Store, Google Play, App Gallery, Galaxy Store, GetApps.
* **Локальные:** RuStore, NashStore, RuMarket.
* **Файлы и Web:** APK, Web-версии.

<Note>
  Магазины, которые **уже добавлены** в ваш проект, пропадают из списка выбора. Нельзя добавить два одинаковых магазина (например, два Google Play) в один проект.
</Note>

## Настройки платформы

Чтобы отредактировать параметры (например, если изменилась ссылка на приложение):

1. В левом меню выберите нужную платформу (например, **App Store**).
2. Нажмите на пункт **Настройки**.

### Поля настроек

<ResponseField name="Идентификатор приложения" type="String">
  App ID или Bundle ID вашего приложения (например, `com.company.app`).
</ResponseField>

<ResponseField name="Ссылка на ваше приложение" type="URL">
  Публичная ссылка на страницу приложения в магазине.
</ResponseField>

<ResponseField name="Видимость платформы" type="Переключатель">
  Позволяет временно скрыть платформу из дашборда, не удаляя её.
  *История версий останется доступной, но сама платформа не будет отвлекать внимание.*
</ResponseField>

<ResponseField name="Интеграция" type="ID">
  Уникальный идентификатор для разработчиков. Используется при настройке API.
  *Поле доступно только для чтения (есть кнопка копирования).*
</ResponseField>

## Удаление платформы

Блок удаления находится в самом низу страницы настроек платформы.

<Warning>
  **Внимание: Необратимое действие!**
  
  Удаление платформы приведет к **потере всех версий (релизов)**, связанных с ней. Восстановить эти данные будет невозможно. Пожалуйста, убедитесь, что у вас есть резервные копии.
</Warning>
